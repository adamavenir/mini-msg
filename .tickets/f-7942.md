# Phase 4: Sync backend setup commands

Implement fray sync setup commands for configuring sync backends.

**Context**: Read docs/MULTI-MACHINE-SYNC-PLAN.md Phase 4.

**Files to create**: internal/command/sync.go

**Commands**:
```bash
fray sync status                 # Show current sync configuration
fray sync setup --icloud         # Configure iCloud sync
fray sync setup --dropbox        # Configure Dropbox sync
fray sync setup --path DIR       # Configure custom sync path
```

**Implementation for cloud sync**:
- Create symlink from .fray/shared/ to cloud storage location
- Store config in fray-config.json

**Example iCloud path**:
~/Library/Mobile Documents/com~apple~CloudDocs/fray-sync/{channel-name}/shared

**Tests required**:
- Test sync status shows configuration
- Test symlink is created for cloud sync
- Test custom path works
